# **App Name**: MediQuest AI

## Core Features:

- AI Medical Report Analysis: Use a tool to extract data from uploaded medical reports using OCR and provide simplified AI explanations, detecting abnormal values and historical comparisons.
- Scan Plate Analysis: Allow users to upload X-ray/CT/MRI images for Vision AI-based anomaly detection, displaying heatmap visualizations and urgency classifications.
- Doctor Discovery and Appointment Booking: Enable location-based doctor search and slot-based appointment booking, including appointment history.
- Medicine Ordering: Allow users to view prescriptions and order medicines from nearby stores with order tracking.
- Patient Profile Management: Store personal and health-related information, maintaining allergy and chronic condition records.
- Secure Authentication and Role Management: Implement secure signup and login with role-based dashboards (Patient, Doctor, Medicine Store) and JWT/Firebase authentication.
- Data Consent Management: Implement features to manage granting and revoking doctor access with time-limited consent and access logs.

## Style Guidelines:

- Primary color: Gentle blue (#64B5F6) to evoke trust and health.
- Background color: Very light blue (#E3F2FD), near white, for a clean, clinical feel.
- Accent color: Muted green (#006400 ) and #F7879A (a softer, classic rose) to represent healing and growth.
- Body and headline font: 'Inter', a sans-serif, to provide a modern, clean and accessible appearance
- Use clear, minimalist icons to represent medical data, actions, and categories. Consistent styling throughout the platform.
- Implement a responsive and intuitive layout, optimized for various screen sizes and devices.
- Use subtle transitions and animations to enhance user experience and provide feedback on interactions.
